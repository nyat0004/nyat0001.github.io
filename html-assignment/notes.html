Essential HTML Tags

h1 to h6
Defines headings, with h1 being the largest and h6 the smallest.
p
Represents a paragraph of text.
img
Embeds an image using the "src" attribute.
a
Creates hyperlinks to other web pages or sections within the same page.
ul and li
Used to create unordered lists with bullet points.
